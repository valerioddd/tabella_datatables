# Table with datatables
Simple table with .NET, JQuery and Datatables.

## Features
- Reorderable rows
- Rows color changing
- Deletable rows
- XML rapresentation download

## Secondary features
- AJAX 
- Datatables features: table creation, rowReorder, Buttons
- Bootstrap modal
